Sexlab More Arousal Penalty
=====================================


これは何？
-------------------------------------
540 名前：ほのぼのえっちさん[sage] 投稿日：2018/08/07(火) 09:29:05.15 ID:???0
Arousedの性欲ペナルティをめっちゃ厳しくしたらどうだろう
設定では性欲が溜まりすぎるとスキル成長-5%だったと思うが集中できないからマジカを極端に落
とすとか回復しなくなるとかの改造は簡単にできそう


実装しました。Arousalを計算するタイミングと、セックスが終わったタイミングで発動します。
PCのみ。性欲値が50, 70, 90の時に、マジカとスタミナの回復速度を破滅的に落とします。


仕様
-------------------------------------
・素のMagickaRate/StaminaRateがModによっていじられることが多いため、付呪などでも増やす
  ことができるMagickaRateMult/StaminaRateMultをいじっています。
・付呪している環境下でも効果を発揮させるために、75%, 125%, 300%落とします。付呪していな
  い環境下では破滅的な下がり幅になります。


設定可能な値
-------------------------------------
50, 70, 90の数値(効果が発生するライン)は、それぞれSSLMAP50, SSLMAP70, SSLMAP90のグロー
バル変数で設定しています。それ以外はxEditでいじってください。
